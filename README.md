# ogo de Quebra-Cabeça (Unity)

## Sobre o projeto
Este projeto foi desenvolvido como um estudo de desenvolvimento de jogos utilizando a Unity, com apoio do ChatGPT.

A proposta foi criar um jogo simples para focar nos fundamentos, como manipulação de objetos, lógica de interação e organização de projeto.

---

## Objetivo
O objetivo do jogo é montar um quebra-cabeça a partir de uma imagem embaralhada.

---

## MVP (Produto Mínimo Viável)
O MVP do projeto consiste em:

- Carregar uma imagem
- Dividir a imagem em peças quadradas
- Organizar as peças em uma grid
- Embaralhar as peças
- Permitir a movimentação/interação do jogador
- Detectar quando o puzzle está completo

---

## Aprendizados
Durante o desenvolvimento, foram praticados conceitos como:

- Manipulação de GameObjects
- Sistema de input (interação com o mouse)
- Organização de scripts
- Lógica de embaralhamento
- Controle de estado do jogo
